Cyberpunk 2077 Best or worst??!!

Since Cyberpunk 2077 was announced with a teaser trailer in 2013, the development of the game started big after The Witcher 3 was released. In June 2019 during the E3 presentation, the first release date was April 16 2020. In January 2020 the release date was postponed to 17 September. The second release date was later moved back to 19 November.

Even though all seemed like the November date was set in stone, CD Projekt RED delayed the game one more time to December 10th 2020. The game has gone gold, this means, that the additional time is used for the day 1 patch. It's not easy to hear, but in the end we have a more round game in hand.
